Administração

Aprender sobre administrar empresas, como funciona um planejamento estratégico de uma empresa, organização, finanças, lideranças e marketing.

Desenvolvimento de Sistemas

Aprender sobre a lógica de programação, compreendendo assim suas linguagens. Além disso, aprender a desenvolver uma boa análise de um sistema, e como harmonizar as cores no design, entre outros aspectos.

Turismo

Aprender a como organizar uma boa viagem, conseguindo assim, escolher um guia turístico atrativo para os clientes, proporcionando uma melhor experiência.

Edificações

Aprender como funciona o planejamento de uma construção de pequeno e/ou grande porte, a importância do tipo de terra para a construção, como fazer uma planta, quais materiais são necessários e dentre outros aspectos.

Enfermagem

Aprender a anatomia do corpo humano, como cada parte funciona para entender quais medicamentos cada pessoa necessita. Além disso, compreender como cada tipo de pessoa pode reagir a um tipo de medicamento.

Ciências Humanas

Construir novos saberes e habilidades para solucionar problemas do mundo real, mobilizando linguagens, raciocínio lógico-matemático, conhecimentos sócio-históricos, científicos, tecnológicos, artísticos e culturais, no contexto da sociedade contemporânea.

Ciências Exatas

Aprender a usar a lógica para resolver problemas complexos, aprender como o mundo funciona, desde os maiores fenômenos envolvendo transformações até os mais básicos.
